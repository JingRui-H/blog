springboot自动装配使用

自定义spring自动加载

@Import

只需要@Import写入一个Enable**注解，之后就可以直接使用默认配置了

使用spring.factories文件

比如resources/META-INF/spring.factories中添加

Condition配置

比如

  • @ConditionalOnProperty：配置文件
  • @ConditionalOnBean({ RedisConnectionFactory.class })：RedisConnectionFactory存在上下文中才会配置
  • @ConditionalOnMissingBean：不存在才配置
  • @ConditionalOnClass、@ConditionalOnMissingClass
  • @ConditionalOnExpression(value = "${a.enable:true} and ${b.enable:true}")：多个配置项符合条件
  • @ConditionalOnSingleCandidate
  • @ConditionalOnResource(resources = "/logback.xml")：我们要查找的bean依赖指定资源是否存在于classpath
  • @ConditionalOnJndi：只有指定的资源通过 JNDI 加载后才加载 bean
  • @ConditionalOnJava：只有运行指定版本的 Java 才会加载 Bean
  • @ConditionalOnWebApplication、@ConditionalOnNotWebApplication：是否在web/非web情况下加载该配置
  • @ConditionalOnCloudPlatform：只有运行在指定的云平台上才加载指定的 bean

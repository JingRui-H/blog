cpu占用100解决

  1. 安装插件choose runtime
  2. 打开Help -> Find Action, 搜索choose runtime
  3. 点击choose runtime就出现Loading Runtime List...的弹出框
  4. 选择本机自己安装的jdk
  5. 重启

感谢https://blog.csdn.net/weixin_44729500/article/details/105041902

记录一下自己比较喜欢的插件， 没办法， 喜欢重装系统， 每次都去百度一遍挺难受。

快捷键 ctrl+alt+s 打开设置， 自行打开插件栏，略

JRebel for IntelliJ

  1. 下载 https://github.com/ilanyu/ReverseProxy/releases/tag/v1.4 这里我下载win64版本ReverseProxy_windows_amd64.exe，点击开启
  2. 安转插件JRebel for IntelliJ
  3. 从新打开idea，点击debug jrebel
  4. 选择默认的Connect to online licensing service, 第一行填写http://127.0.0.1:8888/2aacfda2-91c7-4ca3-bd41-cafe937878e2， 这里可以生成GUID. 第二行输入正确邮箱格式就行， 同意协议点击确定。
  5. 激活成功，打开settings，ctrl+alt+s ，点击JRebel
  6. 点击 work offline， 之后就可以关闭ReverseProxy_windows_amd64.exe了。

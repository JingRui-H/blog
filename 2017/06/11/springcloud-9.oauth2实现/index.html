springcloud-OAuth2实现

OAuth2网上文章比较多, 但是实战写的比较详细的很少, 方法不知道具体什么意思. 在这里先感谢 程序员DD的从零开始的Spring Security Oauth2

代码

pom.xml

OAuth2ServerConfig

这个配置ResourceServer, 哪些资源需要鉴权.

配置oauth客户端相关信息, 他会根据oauth/token请求携带的参数去比对这里配置的参数, 如果正确就认定通过. client_secret是明文传输, 然后会根据配置的加密方式去加密, 比对这边配置的secret.

配置token保存方式和用户名密码鉴别方式userDetailsService

密码加密方式:

  1. 用 BCrypt 对密码编码 String finalSecret = new BCryptPasswordEncoder().encode("123456");
  2. 支持多种编码，通过密码的前缀区分编码方式 String finalSecret = "{bcrypt}" + new BCryptPasswordEncoder().encode("123456");
